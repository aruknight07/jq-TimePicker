Custom TimePicker

You can create time-pickers specifying if you would like them to be in format of 12 hours or 24 hours

A basic way to create a time-picker would be by having a Div (in this case, the ID of the div will be "format12") and add the following line of code

$("#format12").timepicker({ hours: 8, minutes: 0, seconds: 0, mode: "am" });
